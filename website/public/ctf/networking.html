McDonalds

Sockets allow you to talk to other programs over a network! Hosts have IP addresses and ports. You can think of the IP address as the location of the McDonalds, and the ports as the register you're ordering at. Each McDonalds has multiple registers that let's you check out food and order stuff. Similarly, each application running on a host can bind to its own port. The IP 137.151.29.178 on port 1234 has the flag for this challenge. Use a tool like nc to connect!

Simple Arithmetic

You can write code to interface with sockets too! "https://www.geeksforgeeks.org/socket-programming-python/" These can be useful for doing complex, time sensitive tasks with sockets. In a problem like this, the goal will be to connect to the socket, read the data, do something, and send it back! In this case, this socket spits two numbers at you, and wants you to add them really fast. Probably too fast to do by hand- you'll need to write code!
